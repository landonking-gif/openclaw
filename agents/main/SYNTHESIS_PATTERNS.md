# Data Synthesis Patterns — Agentic-3

## Core Synthesis Patterns

### 1. Source Triangulation
**When:** Multiple conflicting sources exist  
**Process:**
1. Identify all source documents
2. Extract key claims from each
3. Map agreements and conflicts
4. Weight by source reliability
5. Document resolution logic

### 2. Narrative Construction
**When:** Raw data lacks coherent story  
**Process:**
1. Identify protagonist/entity
2. Map chronological events
3. Fill gaps with "likely" bridges
4. Flag speculative elements
5. Build causal chains

### 3. Gap Analysis
**When:** Information is incomplete  
**Process:**
1. Document what IS known
2. Explicitly list unknowns
3. Research missing pieces
4. Note confidence levels

### 4. Consolidation
**When:** Multiple reports cover same topic  
**Process:**
1. Deduplicate facts
2. Merge complementary details
3. Resolve contradictions
4. Create unified summary

### 5. Trend Extraction
**When:** Analyzing time-series data  
**Process:**
1. Identify metrics
2. Calculate deltas
3. Spot patterns
4. Project forward
5. Note contributing factors

## Quality Indicators
- ✅ Multiple independent sources confirm
- ⚠️ Single source / unverified
- ❌ Contradicted by reliable source
