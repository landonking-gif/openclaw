# Automated Checklist Generator

## Usage
Run when creating new files or at end of session.

## Quality Gates
### For Each File
- [ ] Clear purpose stated
- [ ] Proper naming (kebab-case.md)
- [ ] Header includes date
- [ ] Links to related files
- [ ] No temporary/debug content

### For Documentation
- [ ] Table of contents if >500 lines
- [ ] Examples where helpful
- [ ] Status indicators (✅❌⚠️)
- [ ] Last updated timestamp
- [ ] Contact/escalation info

### For Templates
- [ ] Variable placeholders ({{VAR}})
- [ ] Instructions for use
- [ ] Example filled version
- [ ] Validation rules

### For Reports
- [ ] Executive summary
- [ ] Data sources cited
- [ ] Confidence levels noted
- [ ] Recommendations actionable
- [ ] Next steps clear

## Index Maintenance
- [ ] Add to REFERENCE_INDEX.md
- [ ] Update file count
- [ ] Check for orphans
- [ ] Verify links work

## Session Close
- [ ] Update memory/YYYY-MM-DD.md
- [ ] Commit to MEMORY.md if significant
- [ ] Document learnings
- [ ] Plan next actions
