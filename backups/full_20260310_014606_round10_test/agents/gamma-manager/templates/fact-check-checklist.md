# Fact-Check Checklist — agentic-4

## Pre-Check
- [ ] Claim clearly identified
- [ ] Source material gathered
- [ ] Claim context understood

## Verification Steps
1. **Primary Source Check**
   - [ ] Official documentation consulted
   - [ ] Original data source located
   - [ ] Direct quote verified

2. **Cross-Reference**
   - [ ] Multiple independent sources found
   - [ ] Sources agree on key facts
   - [ ] Conflicting info noted

3. **Credibility Assessment**
   - [ ] Author credentials verified
   - [ ] Publication date current
   - [ ] Source bias noted

4. **Conclusion**
   - [ ] Verdict: TRUE / FALSE / PARTIAL / UNVERIFIABLE
   - [ ] Confidence level: HIGH / MEDIUM / LOW
   - [ ] Explanation provided

## Documentation
- Sources cited with URLs
- Dates of access noted
- Conflicts explicitly reported
