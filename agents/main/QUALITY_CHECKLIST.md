# Synthesis Quality Checklist

## Pre-Synthesis
- [ ] All sources gathered
- [ ] Source reliability assessed
- [ ] Scope defined
- [ ] Deadline clear

## During Synthesis
- [ ] Claims verified (where possible)
- [ ] Conflicts noted
- [ ] Uncertainty flagged
- [ ] Logical flow maintained

## Post-Synthesis
- [ ] Executive summary concise
- [ ] Evidence cited
- [ ] Gaps documented
- [ ] Recommendations actionable
- [ ] Confidence level stated
- [ ] Formatting consistent

## Final Review
- [ ] Read aloud for clarity
- [ ] Check for hidden assumptions
- [ ] Verify citations
- [ ] Include uncertainty for weak claims
