# Template Library

## Available Templates

### Session Reports
- `session-summary.md` — End-of-session recap

### Task Management
- `task-tracker.md` — Daily task board

### Research
- `synthesis-report.md` — Findings synthesis

### Agent-Specific
- To be created per agent needs

## Template Guidelines
1. Always use {{variables}} for dynamic content
2. Include date/time fields
3. Add status sections
4. Leave room for notes

## Creating New Templates
1. Create in `TEMPLATES/` folder
2. Use descriptive naming
3. Document purpose
4. Add to this index
