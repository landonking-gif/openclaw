# SKILL.md — Data Synthesis Techniques

## Core Synthesis Patterns

### 1. Multi-Source Reconciliation
- **When:** Conflicting data from multiple sources
- **Method:** Compare claims → Identify agreement points → Flag conflicts → Document confidence levels
- **Output:** Reconciled narrative with source attributions

### 2. Trend Extraction
- **When:** Time-series data or pattern recognition needed
- **Method:** Normalize data → Look for correlations → Identify anomalies → Project forward
- **Output:** Trend summary with strength indicators

### 3. Comparative Analysis
- **When:** Benchmarking or comparing options
- **Method:** Establish criteria → Score each option → Weight importance → Document tradeoffs
- **Output:** Comparison matrix with recommendations

### 4. Narrative Construction
- **When:** Building coherent story from disparate facts
- **Method:** Gather all points → Find logical flow → Fill gaps → Preserve nuance
- **Output:** Flowing narrative that respects source complexity

## Operating Principles
1. **Source first** — Always cite where data comes from
2. **Confidence labels** — Mark certainty: high/medium/low
3. **Preserve dissent** — Don't smooth away valid differences
4. **Quantify when possible** — Numbers over vague language
5. **Actionable outputs** — Synthesis should lead to decisions
