# Summary Templates — General-2
*Reusable templates for common summarization tasks*

## 1. Document Summary (Executive)
```
## TL;DR
[1-2 sentence high-level summary]

## Key Points
- [Point 1]
- [Point 2]
- [Point 3]

## Detailed Summary
[Paragraph format with nuance and context]

## Action Items
- [ ] [Action 1]
- [ ] [Action 2]
```

## 2. Meeting Notes
```
## Meeting: [Title] | [Date]
**Attendees:** [Names]

### Decisions Made
- [Decision 1]

### Action Items
| Owner | Task | Due |
|-------|------|-----|
| [Name] | [Task] | [Date] |

### Notes
[Key discussion points]
```

## 3. Code Review Summary
```
## PR Summary: #[Number]
**Scope:** [What changed]
**Risk Level:** [Low/Med/High]

### Changes
- [Change 1]

### Concerns
- [Issue or none]

### Verdict
[Approve/Request changes/Comment]
```

## 4. Error Log Summary
```
## Error Cluster: [Name]
**Window:** [Time range]
**Count:** [N occurrences]

### Pattern
[What's happening]

### Root Cause
[Hypothesis]

### Fix Needed
[Recommendation]
```

## 5. Agent Activity Report
```
## Status: [Agent Name]
**Time:** [Timestamp]

### Current Activity
[What they're doing]

### Completed (Last 2h)
- [Item 1]

### Planned (Next 2h)
- [Item 1]

### Blockers
- [None or list]
```
